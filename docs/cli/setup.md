# `fastclaw setup`

Initialize `~/.fastclaw/fastclaw.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
fastclaw setup
fastclaw setup --workspace ~/.fastclaw/workspace
```

To run the wizard via setup:

```bash
fastclaw setup --wizard
```
