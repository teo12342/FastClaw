# `fastclaw tui`

Open the terminal UI connected to the Gateway.

Related:

- TUI guide: [TUI](/web/tui)

## Examples

```bash
fastclaw tui
fastclaw tui --url ws://127.0.0.1:18789 --token <token>
fastclaw tui --session main --deliver
```
